# LCNI Data Collector - Version 5.3.9f

## Changes
- Chuyển điều hướng Member thành mục `Member` trong menu `LCNI Data` và thêm tab tắt trong Frontend Setting.
- Nâng cấp cấu hình shortcode `[lcni_member_login]` và `[lcni_member_register]` với bộ tùy chỉnh giao diện chi tiết (input/button/background/icon).
- Tách cấu hình Quote sang tab riêng dùng chung Login/Register, có random quote từ danh sách hoặc CSV (1 dòng/quote).
- Sửa xử lý login shortcode để đăng nhập và redirect đúng URL cấu hình.
